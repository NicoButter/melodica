# Assets Directory

Esta carpeta contiene todos los recursos estáticos de la aplicación Melodica.

## Estructura

- `images/` - Imágenes para la aplicación
  - `home/` - Imágenes para la página Home (hero backgrounds, etc.)
  - `about/` - Imágenes para la página About (fotos de perfil, etc.)
  - `instrumento/` - Imágenes para Guitarra y Piano
  - `acorde/` - Imágenes para Selector de Acordes
  - `layout/` - Imágenes para Header y Footer
  - `shared/` - Imágenes compartidas (logos, icons, etc.)
- `audio/` - Archivos de audio (composiciones musicales)

## Uso en el código

- Imágenes: `url('/assets/images/[componente]/nombre-imagen.jpg')`
- Audio: `src="/assets/audio/nombre-audio.mp3"`
