# Imágenes de la Sección Taller

Este directorio contiene las imágenes utilizadas en la sección "Tu Taller Musical" de la aplicación Melodica.

## Estructura sugerida

- `guitarra-virtual.jpg/png` - Imagen representativa de la guitarra virtual
- `piano-virtual.jpg/png` - Imagen representativa del piano virtual
- `selector-acordes.jpg/png` - Imagen representativa del selector de acordes
- `background-taller.jpg/png` - Imagen de fondo para la sección taller (opcional)

## Especificaciones técnicas

- **Formato**: JPG o PNG
- **Resolución recomendada**: 800x600px mínimo
- **Tamaño**: Máximo 500KB por imagen
- **Estilo**: Alineado con la estética bohemia/moderna de Melodica

## Uso en el componente

Las imágenes se referencian desde el componente taller usando rutas relativas:
```html
<img src="assets/images/taller/guitarra-virtual.jpg" alt="Guitarra Virtual">
```
